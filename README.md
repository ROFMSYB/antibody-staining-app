# 🧬 流式抗体配方计算器

一个基于 Streamlit 的网页工具，用于计算流式抗体染色实验所需配方体积。
支持网页手动输入与 Excel 上传两种方式。

---

## 🚀 功能概览

- ✅ 在线填写或粘贴抗体信息
- 📁 上传标准 `.xlsx` 文件进行计算
- 🔢 自定义样本数量
- 📥 导出包含一抗/二抗/胞内抗体的 Excel 结果

---

## 🧪 快速开始

### 1) 安装依赖

```bash
pip install -r requirements.txt
```

### 2) 启动应用

```bash
streamlit run app.py
```

浏览器默认打开：`http://localhost:8501`

---

## 📋 数据输入字段

| 字段名 | 示例 | 必填 | 说明 |
|---|---|---|---|
| marker | CD3 | ✅ | 抗体标记名称 |
| 荧光染料 | FITC | ✅ | 荧光标签名称 |
| 稀释比例 | 1:100 | ✅ | 必须为 `1:数字` 格式 |
| 是否作为FMO | 是 / 空 | ✳️ | 填“是”表示参与 FMO |
| 一抗/二抗/胞内抗体 | 一抗 | ✅ | 支持：一抗/二抗/胞内抗体/自发荧光 |

✳️ 留空默认不作为 FMO。

---

## 📤 输出结果

点击“开始计算”后会生成 `staining_result.xlsx`，可通过页面按钮下载。

## 🛠️ 项目结构

- `app.py`：Streamlit 页面与交互
- `staining_logic.py`：输入校验、标准化、计算与导出逻辑
- `3_edition.py`：命令行兼容脚本（调用同一套核心逻辑）
